# Notification Enhancements

## Requirements

### 1. 通知类型筛选 Tab

- 顶部显示 tab 栏：全部 / 点赞 / 评论 / 关注 / 收藏
- 切换 tab 时列表实时过滤，分组标签（今天/昨天/更早）保持正常显示
- 当前 tab 有未读通知时，tab 上显示未读数角标

### 2. 单条标记已读

- 每张通知卡片右上角提供"标记已读"按钮（仅未读时显示）
- 点击后该条 isRead 变为 true，不影响其他条目
- 需要调用后端 API（或复用现有 markNotificationsRead 逻辑扩展为单条）

### 3. 导航入口未读角标

- 在侧边栏 / 底部导航的通知入口上显示未读总数
- 未读数为 0 时角标隐藏
- 标记已读后角标实时更新

### 4. 新通知淡入动画

- 新进入列表的通知卡片触发 fade-in + slide-down 动画
- 动画时长约 250ms，使用 CSS transition

### 5. 分页 / 无限滚动

- 初始加载最新 20 条
- 滚动到底部自动加载下一页
- 加载中显示 spinner，无更多数据时显示提示文字
